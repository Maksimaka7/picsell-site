Збір та використання персональних даних

Типи даних, що збираються

Під час використання нашого Сервісу ми можемо попросити Вас надати нам певну особисто ідентифікаційну інформацію, яка може бути використана для зв'язку з Вами або ідентифікації Вас. Особисто ідентифікаційна інформація може включати, але не обмежується:

  • Ім'я та прізвище
  • Номер телефону
  • Дані про використання

Дані про використання

Дані про використання збираються автоматично під час використання Сервісу. Вони можуть включати таку інформацію, як IP-адреса Вашого пристрою, тип браузера, версія браузера, сторінки нашого Сервісу, які Ви відвідуєте, час та дата Вашого візиту, час, проведений на цих сторінках, унікальні ідентифікатори пристроїв та інші діагностичні дані.

Інформація, зібрана під час використання Додатку

Під час використання додатку PICSELL ми можемо збирати, з Вашого попереднього дозволу:

  • Інформацію про місцезнаходження: щоб визначити, чи дійсно користувач знаходиться у бажаному місці, додаток PICSELL відстежує місцезнаходження навіть коли додаток закрито або не використовується. Збір даних про місцезнаходження здійснюється тільки під час візиту. Приймаючи угоду користувача, ви даєте згоду на використання Вашого місцезнаходження для ділових цілей: робота з маршрутами та пошук торгових точок, відстеження активного робочого часу мерчандайзера.
  • Фотографії та інша інформація з камери та фотобібліотеки Вашого пристрою.

Використання персональних даних

Компанія може використовувати Персональні дані для таких цілей:

  • Для управління Вашим Обліковим записом.
  • Для виконання контракту: розробка, дотримання та виконання договору купівлі-продажу.
  • Для зв'язку з Вами електронною поштою, телефонними дзвінками, SMS або іншими формами електронного спілкування.

  • Для бізнес-передач у разі злиття, придбання або продажу активів.
